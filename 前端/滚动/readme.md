# 常用滚动/虚拟滚动库分类

## 悬浮滚动条
- [handy-scroll]

## 🎨 自定义滚动条（美化/替换原生滚动条）
- [perfect-scrollbar](https://perfectscrollbar.com/)：轻量、无依赖，提供自定义滚动条及平滑滚动。
- [simplebar](https://grsmto.github.io/simplebar/)：将原生滚动条替换为自定义样式，支持响应式。
- [OverlayScrollbars](https://kingsora.github.io/OverlayScrollbars/)：功能丰富的自定义滚动条插件，样式高度可配置。
- [overlayscrollbars-vue]：vue3 OverlayScrollbars 版
- [smooth-scrollbar](https://idiotwu.github.io/smooth-scrollbar/)：可定制平滑滚动效果，支持插件扩展。

## ⚡️ 虚拟滚动（高效渲染长列表）
- [vue-virtual-scroller](https://github.com/Akryum/vue-virtual-scroller)：Vue 专用组件，适合列表/网格/表格的虚拟滚动。
- [virtual-scroll](https://github.com/ayamflow/virtual-scroll)：轻量虚拟滚动库（通常与框架无关），处理海量列表渲染。

## 📜 通用滚动容器/交互库
- [iScroll](https://github.com/cubiq/iscroll)：老牌移动端滚动库，支持平滑滚动、缩放、下拉刷新、自定义滚动条等。

## 🧭 滚动触发行为（按滚动方向控制UI）
- [headroom.js](https://wicky.nillia.ms/headroom.js/)：监听滚动位置，自动隐藏或显示页面头部。

## 🌊 瀑布流布局（滚动式网格布局）
- [vue-waterfall](https://github.com/MopTym/vue-waterfall)：Vue 的瀑布流组件，基于滚动动态排列图片/卡片。

# 理论 / CSS 属性

- `scrollbar-gutter`：控制滚动条与内容之间的间距
- `scrollbar-width: thin;`：设置滚动条宽度为细
