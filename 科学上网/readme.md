# V2Ray客户端

下面是常见的 V2Ray / Clash 类客户端，按系统分类：

## Windows
- v2rayN：最常用的 Windows V2Ray 客户端，配置简单，支持订阅和手动导入。
- Clash Verge：基于 Clash 的图形客户端，界面友好，适合长期使用。
- Clash for Windows（CFW）：老牌 Windows 客户端，功能全面。
- Nekoray / Qv2ray：适合喜欢更灵活配置的用户。

## macOS
- ClashX：macOS 上非常常见的 Clash 客户端，使用体验较好。
- Clash Verge：也支持 macOS，可作为通用选择。
- V2RayU：老牌 macOS V2Ray 客户端，适合传统配置方式。
- V2RayX：早期常见的 macOS V2Ray 图形客户端。
- ShadowsocksX-NG：如果你主要用 Shadowsocks 协议，也可以作为备选。

## 选择建议
- Windows：优先试 v2rayN 或 Clash Verge。
- macOS：优先试 ClashX 或 Clash Verge。
- 如果你只想快速上网，选有订阅功能的客户端更省心。
