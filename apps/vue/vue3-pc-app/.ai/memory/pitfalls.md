# 踩坑记忆

这个文件记录本项目中容易重复发生的问题。

## 已知坑点

### `src/views` 是页面目录

harness 示例中可能使用 `src/pages`，但本仓库当前使用 `src/views`。如果没有整体迁移计划，不要单独新增 `src/pages`。

### 路由和侧边栏需要同步

顶层路由定义在 `src/router/index.ts`，侧边栏菜单定义在 `src/App.vue`。只改其中一个，会出现隐藏页面或无法导航的问题。

### 脚手架测试可能落后于真实 UI

初始测试里可能还在断言 `You did it!` 这类脚手架文案。修改应用壳层或默认路由后，要让测试匹配真实界面。

### Element Plus 实例 ref 需要类型

Element Plus 组件模板 ref 通常初始为 `null`。当调用 `setScrollTop` 等组件方法时，需要补充合适类型。

## 如何更新

当发现某个错误未来很可能再次发生时，在这里追加一条。每条保持简短，说明症状和推荐修复方式。

## 文件作用

用这个文件回答：“这个项目之前有哪些容易踩的坑？”
