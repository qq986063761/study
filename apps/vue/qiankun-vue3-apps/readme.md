# qiankun-vue3-demo

Vue3 + Vite + qiankun micro frontend demo with one host app and two sub apps.

## Apps

- `main`: host app, runs on `http://localhost:5173`
- `app1`: qiankun sub app, runs on `http://localhost:5174`
- `app2`: qiankun sub app, runs on `http://localhost:5175`

## Usage

```bash
npm run install:all
npm run dev
```

Open `http://localhost:5173`, then use the left menu to visit `首页`, `App1`, and `App2`.
