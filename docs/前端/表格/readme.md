[表格组件分类]

- **企业级数据网格**: 功能全面，适合复杂数据表格场景
	- [AG-Grid](https://www.ag-grid.com/) — 企业级数据网格，支持多框架、虚拟滚动、行列编辑、聚合等

- **电子表格 / 可编辑网格**: 类似 Excel 的编辑体验
	- [Handsontable](https://handsontable.com/) — 可编辑的电子表格组件
	- [jSpreadsheet](https://jspreadsheet.com/) — 轻量级电子表格风格组件

- **传统表格插件**: 基于 jQuery 的经典方案
	- [DataTables](https://datatables.net/) — 强大的表格增强插件，排序/搜索/分页

- **现代/头部无 UI（Headless）表格库**: 提供逻辑层，易于与 UI 结合
	- [TanStack Table](https://tanstack.com/table/latest) — 现代化的无 UI 表格构建库，适合 React 等框架
  - [TanStack Virtual](https://tanstack.com/virtual/latest)
