# 溢出隐藏

## 文本溢出隐藏资源分类

- 纯 JS 通用文本截断
  - [Superclamp](https://www.npmjs.com/package/superclamp?utm_source=chatgpt.com)
  - [clamp-js](https://www.npmjs.com/package/clamp-js)

- 轻量文本截断插件
  - [Cuttr.js](https://cuttr.kulahs.de/?utm_source=chatgpt.com)

- 现代前端生态辅助库
  - [@chenglou/pretext](https://www.npmjs.com/package/@chenglou/pretext)

## 说明

- `Superclamp`：通用 JavaScript 文本截断 / 多行 clamp 工具，适合浏览器中对普通文本块做多行省略隐藏。
- `clamp-js`：纯 JS 文本截断库，适合作为 line-clamp fallback。
- `Cuttr.js`：轻量文本截断插件，快速实现文字溢出省略、块级文本截断。
- `@chenglou/pretext`：现代前端文本截断/换行辅助库，适合 React/Preact 或现代 JS 项目。
