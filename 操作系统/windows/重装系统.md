## 联网前

### 跳过联网

- `Shift + F10` 弹出终端 → 输入 `OOBE\BYPASSNRO` → 回车

### 个性化

- **背景**：个性化 → 设置背景 → 选图片
- **锁屏界面**：个性化锁屏界面 → 选图片
- **锁屏界面**：个性化锁屏界面 → 关闭「在锁屏界面上获取花絮、提示、技巧等」

### 系统

- **通知**：关闭通知
- **隐私和安全性 → 建议和优惠**：「设置」中的建议和优惠 → 关闭
- **电源和电池 → 电源模式**：最佳性能
- **电源和电池 → 盖子、电源和睡眠**：按电源按钮将使我的电脑 → 关机
- **存储 → 高级存储设置**：保存新内容的位置

### 资源管理器

- 左侧快捷目录 → 属性 → 位置 → 改到其他盘

### Windows 更新

- **高级选项 → 传递优化**：允许从其他设备下载 → 关闭

### 永久关闭 Windows 更新

- **服务 → Windows 更新**：启动类型 → 禁用
- **服务 → Windows 更新**：服务状态 → 停止
- **服务 → Windows 更新 → 恢复**：可下拉选项 → 全部改为「无操作」
- **注册表**：`计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsUpdate\UX\Settings` → 新建 DWORD（32 位）→ 命名 `FlightSettingsMaxPauseDays` → 十进制 `3650`（约可暂停 10 年）→ 回到 Windows 更新 → 暂停更新选最大值
