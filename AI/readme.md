# ai 优点
- 代码结构，逻辑布局上很强，适合利用它做组件的代码布局，功能实现
- 问题分析能力可以，可引导我们找出代码上的问题
- agent 代理人执行流程化全面，比如在编写功能时系统性新增、修改关联文件，翻译上自动补全缺失资源

# ai 当前不足
- 设计的高宝还原度在细节上不够，界面还原上目前不是使用重心

# ai 实践
- 定 cursor skills 规则应用到项目，css、js、html
- 自己找不到问题，可以让 ai 分析可能原因
- 让 ai 根据中文生成国际化配置
- 后端接口数据让 ai 渲染到视图
- 时间紧如果先做了功能，把需求给 ai 审查缺什么没做
- 需求文档直接给 ai 生成区块代码
- 找历史git记录，让 ai 还原部分内容
- tab 快速生成组件引入到使用代码，tab 快速生成个别国际化配置
