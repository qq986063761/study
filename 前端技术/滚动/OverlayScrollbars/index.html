版本 2.15.1 · 精致滚动条 | 纯滚动容器演示

🎯 原生级滚动体验

OverlayScrollbars v2.15.1 提供完全可自定义的滚动条，不破坏原有布局，支持鼠标、触摸板和键盘操作。

🧊 平滑透明叠加层

滚动条以悬浮形式出现，离开容器后自动淡隐，不影响内容区域宽度，视觉更现代。

🎨 主题自适应

当前使用 os-theme-dark 暗色滚动条，适合浅色背景，也可通过 CSS 变量快速换肤。

📜 丰富的滚动配置

支持点击轨道跳转、拖动滑块、滚动惯性增强，完全适配复杂业务场景。

⚡ 性能出色

仅 25KB gzip，无外部依赖，滚动时内存与 CPU 占用极低。

📱 移动端友好

触摸滚动流畅，且支持原生越界回弹效果（需配合 -webkit-overflow-scrolling）

🚀 第7块内容 — 滚动条可见策略

配置了 autoHide: 'scroll'，滚动时显示滚动条，静止后优雅消失，最大化利用空间。

🏔️ 最后一块长内容

确保滚动条出现且交互顺滑。试试用鼠标滚轮快速滚动、或拖动右侧滚动条滑块。
